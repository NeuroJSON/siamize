# tests/

Bundled test data for siamize regression and CI smoke checks.

## Files

| File | Source | License |
|---|---|---|
| `sub-01_T1w.nii.gz` | [OpenNeuro ds000001 v1.0.0](https://openneuro.org/datasets/ds000001/versions/1.0.0), the `sub-01` anatomical T1-weighted scan. | **CC0** (public domain). Compatible with siamize's Apache-2.0 license. |
| `pred_ref_allfolds.nii.gz` | Generated by running `py/siam_ref.py --folds all` on `sub-01_T1w.nii.gz` (5-fold ensemble). Voxel-level regression target for the ONNX and C++ pipelines. | Apache-2.0 (derived work of this repo, against CC0 input). |

## How to regenerate the reference

```bash
python3 py/siam_ref.py \
    -i tests/sub-01_T1w.nii.gz \
    -o tests/pred_ref_allfolds.nii.gz \
    --folds all -v
```

Requires PyTorch and the SIAM v0.3 weights at
`$HOME/siam_params/v0.3/pred_DS108_LcsfP_Ano` (or set `--model-dir`).
